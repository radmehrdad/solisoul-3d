# SoliSoul 3D

An immersive Persian-language mental-health website built with HTML, CSS, JavaScript, and Three.js.

## Live site

https://radmehrdad.github.io/solisoul-3d/

## Run locally

Serve the repository with any static HTTP server, for example:

```bash
python3 -m http.server 8765
```

Then open `http://127.0.0.1:8765/`.
